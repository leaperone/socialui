# UI2SOMEone 组件库

> UI2SOMEone 是一个专为社交媒体链接与二维码展示场景设计的 React 组件库，帮助你快速集成美观、统一的社交分享能力。基于 Tailwind CSS 与 daisyUI，支持多平台社交卡片、按钮、二维码等常用场景，助力高效开发和一致的用户体验。

---

## Vision

说到组件库，往往都是按钮、输入框、表格、弹窗等基础组件，而难见到其他常用场景的复用组件。
例如公众号后台提供的分享物料，就是一个典型的场景。公众号提供几种样式的图片以及 Logo 等的工程文件。而对于有开发能力的人来说，真的需要在网页平台上面上传各种平台的分享物料图片吗？

于是我们设计了 UI2SOMEone 组件库，希望 UI2SOMEone 能够成为社交媒体链接与二维码展示场景的标配组件库，让开发者能够快速集成各类社交卡片、按钮、二维码等组件，让你的产品快速具备美观、统一的社交分享能力。

## 🎨 主题与定制

- 组件基于 Tailwind CSS 与 daisyUI，支持自定义主题和样式扩展
- 可通过 Tailwind 配置和 daisyUI 主题系统自定义品牌色、风格
- 支持 className 透传，便于灵活调整样式

## ❓ 常见问题（FAQ）

**Q: 组件库支持哪些社交平台？**  
A: 微信、微博、B 站、Facebook、Line、LinkedIn、Twitch、YouTube 等。

**Q: 如何自定义主题或样式？**  
A: 通过 Tailwind CSS 和 daisyUI 主题系统，或直接传递 className。

**Q: 如何反馈问题或贡献代码？**  
A: 欢迎通过 GitHub Issues 或 Pull Requests 参与贡献。
