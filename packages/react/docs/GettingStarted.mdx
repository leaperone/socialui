# 快速开始

> 当前版本：`v1.2.0`

欢迎使用 UI2Someone！这是一个专为社交媒体链接与二维码展示场景设计的现代化 React 组件库。

## 📦 安装

首先，在你的 React 项目中安装 UI2Someone：

```bash
# 使用 npm
npm install @ui2someone/react

# 使用 pnpm
pnpm add @ui2someone/react

# 使用 yarn
yarn add @ui2someone/react
```

## 🎨 样式配置

UI2Someone 基于 Tailwind CSS 和 DaisyUI 构建。你需要在项目中配置这些依赖：

### 1. 安装样式依赖

```bash
npm install tailwindcss daisyui
```

### 2. 配置 Tailwind CSS

在你的 `tailwind.config.js` 中添加 DaisyUI 插件：

```js
/** @type {import('tailwindcss').Config} */
module.exports = {
  content: ["./src/**/*.{js,ts,jsx,tsx}", "./node_modules/@ui2someone/react/dist/**/*.{js,mjs}"],
  theme: {
    extend: {},
  },
  plugins: [require("daisyui")],
  daisyui: {
    themes: ["light", "dark"],
  },
};
```

### 3. 引入样式

在你的主 CSS 文件中引入 Tailwind CSS：

```css
@tailwind base;
@tailwind components;
@tailwind utilities;
```

## 🚀 基础用法

### 导入组件

```tsx
import { WeChatContactCard, BilibiliProfileCard, XProfileCard } from "@ui2someone/react";
```

### 简单示例

```tsx
import { WeChatContactCard } from "@ui2someone/react";

function App() {
  return (
    <div className="p-4">
      <WeChatContactCard wechatId="your-wechat-id" displayName="我的微信号" />
    </div>
  );
}
```

## 📱 常用组件示例

### 社交媒体卡片

```tsx
import {
  WeChatContactCard,
  XProfileCard,
  BilibiliProfileCard,
  LinkedInProfileCard,
} from "@ui2someone/react";

function SocialCards() {
  return (
    <div className="grid grid-cols-1 md:grid-cols-2 gap-4 p-4">
      {/* 微信联系卡片 */}
      <WeChatContactCard wechatId="your-wechat-id" displayName="联系我" />

      {/* X (Twitter) 个人资料卡片 */}
      <XProfileCard username="your-twitter-handle" displayName="关注我的 X" />

      {/* B站个人资料卡片 */}
      <BilibiliProfileCard uid="123456" username="your-bilibili-name" />

      {/* LinkedIn 个人资料卡片 */}
      <LinkedInProfileCard
        profileUrl="https://linkedin.com/in/yourprofile"
        displayName="我的 LinkedIn"
      />
    </div>
  );
}
```

### 微信官方账号卡片

```tsx
import { WeChatOfficialAccountCard } from "@ui2someone/react";

function OfficialAccount() {
  return (
    <div className="max-w-md mx-auto p-4">
      <WeChatOfficialAccountCard
        qrCodeUrl="/path/to/qr-code.png"
        accountName="我的公众号"
        description="关注获取最新资讯"
      />

```

### 电商卡片

```tsx

}
```

## 🎨 自定义样式

所有组件都支持 `className` 属性来自定义样式：

```tsx
import { WeChatContactCard } from "@ui2someone/react";

function CustomStyledCard() {
  return (
    <WeChatContactCard
      wechatId="your-wechat-id"
      className="shadow-lg border border-gray-200 rounded-xl"
      displayName="自定义样式的微信卡片"
    />
  );
}
```

## 🌙 主题支持

组件支持 DaisyUI 的主题系统：

```tsx
function ThemedApp() {
  return (
    <div data-theme="dark" className="min-h-screen bg-base-100">
      <WeChatContactCard wechatId="your-wechat-id" displayName="暗色主题微信卡片" />
    </div>
  );
}
```

## 📱 响应式设计
